# mylinks-advluana

Este repositório contém a página de "Link na Bio" da Dra. Luana Lira, advogada especialista em Direito Previdenciário em Ferraz de Vasconcelos, SP. A página serve como um hub central para seus contatos e informações online, facilitando o acesso a agendamentos, redes sociais e localização do escritório.

## Tecnologias Utilizadas

*   **HTML5:** Para a estrutura e conteúdo da página.
*   **CSS3:** Estilização in-line e importação de fontes do Google Fonts (`DM Sans`).
*   **SVG:** Ícones para links de contato (WhatsApp, Instagram, Google Maps).
*   **JSON-LD (Schema.org):** Dados estruturados para melhorar a visibilidade em motores de busca, descrevendo o serviço jurídico.

## Estrutura do Projeto

*   `index.html`: O arquivo principal da página, contendo todo o HTML, CSS e metadados.
*   `assets/`: Diretório que armazena recursos como imagens e possivelmente outros arquivos de mídia. No momento, contém imagens de logo e perfil.
*   `robot.txt`: Arquivo que informa aos rastreadores de motores de busca quais páginas devem ou não ser indexadas.

## Funcionalidades Principais

*   **Informações de Contato:** Links diretos para agendamento via WhatsApp.
*   **Presença Online:** Acesso rápido ao perfil do Instagram.
*   **Localização do Escritório:** Link para visualizar o endereço no Google Maps.
*   **Otimização para Buscadores (SEO):** Metatags detalhadas e Schema.org para melhor indexação e exibição nos resultados de busca.
*   **Responsividade:** Design adaptável para diferentes tamanhos de tela.
